# ルールメモ

このアプリのルール本体は `stamp-rules.js` にあります。

## 参加スタンプ

- 対象: 会への参加
- 1巡目の達成条件: 10回
- 1巡目の花: コスモス
- 2巡目の花: 藤
- 3巡目の花: 金木犀
- 1巡ごとに妖精を達成します。
  - 10回: コスモスの妖精
  - 20回: ダリアの妖精
  - 30回: 梅の妖精
- 達成名: コスモス満開
- 勲章: コスモス満開勲章
- 称号: コスモス満開の友

## 先生ごとの妖精達成

同じ先生の指導碁スタンプを5回集めると、その巡に対応する妖精を達成します。
先生ごとに15回まで記録できます。
1巡目が終わった後は、同じ先生カード内で2巡目・3巡目の花が先生ごとに別の花へ変わります。
各先生ごとに、1巡目・2巡目・3巡目の3妖精を達成できます。

## 判定関数

`stamp-rules.js` に以下の関数があります。

- `evaluateParticipationAchievement(count)`
- `evaluateTeacherFairyAchievements(teacherLessonCounts)`
- `evaluateTeacherCircleAchievements(teacherCircleRounds)`
- `evaluateAllAchievements(progress)`

`script.js` 側では、現在の画面上の仮スタンプ数から `evaluateAllAchievements()` を呼んで、冒険者カードに結果を表示しています。

## スタンプを減らしたとき

- 運営がスタンプ数を減らしても、現在のスタンプ数だけを修正します。
- 一度獲得した妖精、勲章、称号、特別な仲間、花図鑑の発見は「冒険の記憶」として残します。
- 減算後に同じ達成条件へ到達し直しても、同じ報酬は重複して保存しません。
- 減算操作は、ほかの管理画面調整と同じく操作履歴へ記録します。
- 明示的な「記録をリセット」は全記録を最初からやり直す操作なので、獲得済み報酬も消去します。

## `userProgress` の形

保存する進捗データは以下の形に整理しています。

- `schemaVersion`
- `stamps.participationCount`
- `stamps.teacherLessonCounts`
- `stamps.teacherCircleRounds`
- `earned.fairies`
- `earned.medals`
- `earned.titles`
- `earned.companions`

古い保存形式が `localStorage` に残っていても、読み込み時にこの形へ寄せます。

## バックアップ形式

- アプリ識別子: `suiyoukai-stamp-adventure`
- バックアップ形式版: `1`
- 保存日時: `savedAt`
- スタンプ数と獲得済み報酬: `progress`
- 操作履歴: `operationHistory`
- 運営パスコード、対局記録、参加者の個人情報は含めません。
- 復元は内容確認と確認語「復元」を必須とし、実行前に現在の状態を自動バックアップします。

## 操作履歴

- 端末内へ最大50件、運営画面には新しい順に10件表示します。
- 各履歴は `recordedAt`（日時）、`type`（操作種別）、`target`（対象）、`before`（変更前）、`after`（変更後）を持ちます。
- 操作種別は、押印、減算、管理調整、復元、全リセットです。
- 管理画面で複数項目を変更した場合は、変更した対象ごとに履歴を追加します。
- 間違いを修正しても以前の履歴は書き換えず、減算または管理調整を新しい履歴として追加します。
- 復元時はバックアップ内の履歴を戻した後、その末尾へ復元操作を追加します。
- パスコード、参加者の個人情報、対局の手合いや勝敗は保存しません。

## 妖精案内人

- 地図の「次の冒険」カード右側に、72pxの小さな案内人を表示します。
- 妖精をまだ獲得していない場合は、収集対象ではない専用の最初の案内役 `fairy-apollon-guide.png` を表示します。
- 獲得済み妖精がいる場合は、`earned.fairies` の最後の妖精を表示します。
- 未獲得の妖精を次の目標として先に表示しません。
- セリフは参加、先生、先生の輪、全達成の4種類の短文だけです。
- 案内人にはタップ判定を持たせず、「次の冒険」カードの操作を妨げません。
- `prefers-reduced-motion: reduce` の場合は浮遊表現を停止します。
