# 8/30イベント 当日運営1枚メモ

状態: 正式保存

初めて触る人に、最初の1回で「今日の記録が花として残った」と見せるための手順。

## 1. 開始前

PC側:

1. 確認用サーバーを起動する。
2. PCで `http://192.168.128.168:4184/` が開けることを確認する。
3. iPhoneでも `http://192.168.128.168:4184/` を開く。
4. 管理画面を開き、パスコードで解除する。
5. 必要なら「記録をリセット」で初期状態にする。

注意:

- URLは必ず `:4184/` まで入れる。
- PCとiPhoneは同じWi-Fiに接続する。
- iPhoneで開けない時は `docs/mobile-preview-connection-runbook-2026-07-04.md` を見る。

## 2. 参加スタンプを反映

1. 参加者に参加フォームを送信してもらう。
2. 送信後はSafariの戻るではなく、ホーム画面から水曜会アプリを開き直してもらう。
3. 参加フォームの回答を確認する。
4. 回答確認はGoogleフォームの回答画面、または回答スプレッドシートで見る。
5. 管理画面の「参加フォーム確認」を見る。
6. 名前を入れる。
7. 「今日の参加スタンプを反映」を押す。

確認すること:

- 参加スタンプは同じ日に1回だけ。
- 反映済みならボタンは押せない。

## 3. 先生スタンプを反映

1. 参加者に対局記録フォームを送信してもらう。
2. 送信後はSafariの戻るではなく、ホーム画面から水曜会アプリを開き直してもらう。
3. 対局記録フォームの回答を確認する。
4. 回答確認はGoogleフォームの回答画面、または回答スプレッドシートで見る。
5. 管理画面の「対局記録フォーム確認」を見る。
6. 先生、対局日、ハンデ、勝敗を選ぶ。
7. 本人スマホへ入れる時は「反映用QRを作る」を押す。
8. 参加者本人のスマホでQRを読み取ってもらう。
9. 冒険者カードの「今日の記録」に着地したことを見る。

確認すること:

- 同じQRをもう一度読んでも二重反映されない。
- 運営PCで確認したい時だけ「対局記録と先生スタンプを反映」を押す。
- 運営PCで確認反映した分は「直前の先生スタンプを取り消す」で戻せる。
- QRで本人スマホに入れた分を間違えた時は、そのスマホ側で管理画面を開いて調整する。

## 4. 参加者に見せる

冒険者カードへ移動したら、画面は「今日の記録」へ着地する。

見せる言葉:

> ここに今日の参加スタンプと先生スタンプが入りました。

フォーム送信後に伝える言葉:

> 戻るボタンではなく、ホーム画面から水曜会アプリを開き直してください。

見せる場所:

- 参加スタンプ: 花マーク
- 指導碁スタンプ: 先生の花スタンプ台紙
- 先生スタンプは1日3枠まで見える

先生の花を押すと、その先生の写真画面へ戻れる。

## 5. 間違えた時

先生を間違えた:

1. 管理画面へ戻る。
2. 運営PCで確認反映した分なら「直前の先生スタンプを取り消す」を押す。
3. QRで本人スマホに入れた分なら、そのスマホ側の管理画面でスタンプ数を調整する。
4. 正しい先生を選び直す。
5. もう一度「反映用QRを作る」を押し、本人スマホで読み取る。

参加スタンプを間違えた:

1. 管理画面の「スタンプ数を調整」で直す。
2. 「確定して保存」を押す。
3. 直近の操作履歴を見る。

不安な時:

- 「直近の操作履歴」を見る。
- 参加者へ見せる前に、冒険者カードの「今日の記録」を確認する。

## 6. 終了後

1. 管理画面で「バックアップを保存」を押す。
2. 保存したファイルを分かる場所へ置く。
3. 必要なら確認用サーバーを止める。
