# 水曜会プロジェクト 引き継ぎメモ

**Ver.1｜2026年8月｜最新版・レビュー用**

## プロジェクトの目的

指導碁会、交流会、体験会などに共通して使える「水曜会らしいポスターの基礎」を作り、今後のイベントを通して育てる。

8月30日のイベントまでは水曜会アプリ本体を大きく変更しない。このプロジェクトではポスター関連のファイルだけを扱い、既存アプリには反映しない。

## 承認済み事項

- ブランドの中心は「優しい」「品がある」「初めての方にも開かれている」。
- 高級感ではなく、肩肘を張らない「温かさのある品」を目指す。
- 3m先からイベント名が分かる構成にする。
- すべてのポスターに「水曜会らしい一言」を入れる。
- 写真は、満員感より初参加者が入る余地を感じられるものを選ぶ。
- QRコードは右下、申込欄はその隣を基本とする。
- 棋士名を写真の上に置くA案をテンプレートの軸とする。
- 色は生成り、深い紺、落ち着いた金茶を基本とする。
- イベント情報欄は、表よりも柔らかなカード表現にする。
- イベント名の上には「指導碁会」「交流会」「体験会」など、種類が分かる分類名を置く。
- ファイルは「完成版」ではなく「最新版」と呼び、版を重ねて育てる。
- 制作はワイヤーフレーム→デザイン見本→使用例の順にレビューする。
- 公開・GitHub・既存アプリへの反映は、悦子さんの明確な承認前には行わない。
