# 参加者試用 2名比較 2026-06-22

## 共通して見つかった問題

1. 花図鑑に花が見えず、どこにあるか・集めるとどうなるか分からない。
2. 文字が小さく、薄い色の文章が読みにくい。
3. 押せる場所と、押しても動かない装飾・表示の区別が分かりにくい。
4. 先生写真の見え方に違和感がある。
5. 妖精やフクロウに興味を持って押すが、期待した反応がない。

## 年代による違い

- 70代男性は、トップ画面から最後まで操作入口を見つけにくかった。
- 20代女性は「まずここを押す」を見つけ、花図鑑へ入れた。
- 20代女性は、妖精のかわいさや次巡の報酬に強く興味を示した。
- 70代男性は地図の到着マークを繰り返し押し、直感的な入口として捉えていた。

## 見直しの優先順位

1. 基本文字を大きくし、薄い文字色のコントラストを上げる。
2. 花図鑑に花を安全に飾り、「集めるとどうなるか」を短く示す。
3. 押せる場所をボタンとして明確にし、押せない表示と区別する。
4. 先生写真のぼけ・切れ・加工状態を実機で見直す。
5. 冒険者カードの妖精を整理し、獲得済み妖精を大きく眺められる表示を加える。
6. 地図の到着マークを「次の冒険」の入口として活用するか検討する。
7. フクロウ画面の役割と、押した時の反応を小さく補う。

## 方針

2名に共通した問題から先に直す。大きな機能追加は避け、文字、花の見え方、操作入口、先生写真を先に改善する。妖精拡大は参加者の好意的反応を生かす小規模な追加として、その次に行う。
