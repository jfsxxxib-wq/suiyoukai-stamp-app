## 参加者にしてもらうこと

最初に読むQR:

- ホーム画面に追加する時は「アプリ入口QR」を使う。
- 受付用 参加QRや先生QRは、花を入れるためのQRなので、ホーム画面追加には使わない。

iPhoneの場合:

1. アプリ入口QRを読み取る。
2. Safariで開く。
3. 共有ボタンを押す。
4. 「ホーム画面に追加」を押す。
5. 名前が `水曜会 花記録` になっていることを確認する。
6. 「追加」を押す。
7. ホーム画面のアイコンから開く。

Androidの場合:

1. アプリ入口QRを読み取る。
2. Chromeで開く。
3. 右上のメニューを開く。
4. 「ホーム画面に追加」または「アプリをインストール」を押す。
5. 名前が `水曜会 花記録` になっていることを確認する。
6. 「追加」または「インストール」を押す。
7. ホーム画面のアイコンから開く。
