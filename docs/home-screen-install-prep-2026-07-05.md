# ホーム画面に入れてもらう準備メモ 2026-07-05

状態: 準備中

目的: 参加者が自分のスマホに「水曜会 花記録」をアプリのように置ける形にする。

## 今回整えたこと

- アプリ名を `水曜会 花記録` にした。
- ホーム画面用アイコンを追加した。
- スマホでホーム画面に追加した時の表示設定を追加した。
- Webアプリ用の `manifest.webmanifest` を追加した。

## 参加者にしてもらうこと

iPhoneの場合:

1. 本番URLをSafariで開く。
2. 共有ボタンを押す。
3. 「ホーム画面に追加」を押す。
4. 名前が `水曜会 花記録` になっていることを確認する。
5. 「追加」を押す。
6. ホーム画面のアイコンから開く。

Androidの場合:

1. 本番URLをChromeで開く。
2. メニューを開く。
3. 「ホーム画面に追加」または「アプリをインストール」を押す。
4. 名前が `水曜会 花記録` になっていることを確認する。
5. ホーム画面のアイコンから開く。

## 本番公開後に必ず確認すること

- GitHub Pagesの本番URLで開ける。
- iPhone Safariでホーム画面に追加できる。
- ホーム画面の名前が `水曜会 花記録` になる。
- ホーム画面のアイコンが花のアイコンになる。
- ホーム画面から開いても、画面が大きく崩れない。
- 受付QR、先生ごとの固定QR、対局内容QRを本番URLで作り直す。
- 本番URLのQRを読んだ後、今日の記録へ着地する。

## まだやらないこと

- 強いオフライン保存やサービスワーカーを入れる。

理由:

- 8/30までは、古い画面がスマホに残る事故を避けたい。
- まずは本番URLで安定して開けることを優先する。
