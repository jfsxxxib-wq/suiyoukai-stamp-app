# 2026-07-06 夜の設計判断メモ

## 参加導線

- 花図鑑の参加スタンプカードは、トップページの参加専用カードと役割が重なったため、花図鑑画面からは非表示にした。
- Android / Apple で、トップページの「水曜会 参加する」から参加専用カードを出し、受付番号を持ってフォームへ進む流れは確認済み。
- フォーム記入だけでは花は入らない。フォーム送信後にホーム画面の花アイコンへ戻り、受付で参加QRを読む流れを参加カード側に集約する。

## 先生写真

- 松本武久先生の写真差し替えを試験実施した。
- 試験表示した写真は若いため、本採用候補ではない。
- 常石先生も同じ理由で差し替えが必要になる可能性がある。
- 先生写真は、年齢、肩書き、掲載許可、画質、日本での見え方や流行により、今後も差し替わる前提にする。
- 先生写真の参照先は `styles.css` の `Current teacher photo sources` に集約した。次回以降は該当先生の画像指定を1か所変える運用にする。

## 管理画面

- 現時点では、管理画面に写真を入れる優先度は高くない。
- 写真が必要になるのは、もっと人数が増えた時、または金銭や厳密な本人確認が関わる時と考える。
- 今は写真よりも、受付番号、画面名、本名または確認名で「誰の記録を操作しているか」を確認できることを優先する。
- QR以外の方法でスタンプ反映、成績反映、修正をする時に、別の人へ記録が付かないことが重要。
- 管理画面上部に「操作対象」カードを追加した。受付番号、画面名、本名・確認名メモを表示し、反映・修正前の確認に使う。
