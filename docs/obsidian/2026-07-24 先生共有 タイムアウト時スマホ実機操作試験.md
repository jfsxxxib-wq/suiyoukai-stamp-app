# 先生共有 タイムアウト時スマホ実機操作試験

- 実施日: 2026-07-24
- 判定: 合格
- 端末: iPhone / Safari
- 接続: PCとスマホを同じWi-Fiに接続したローカル試験
- 公開・GitHub保存・コミット・プッシュ: 未実施

## 結果

1. 花と今日の記録の端末内保存: 合格
2. 15秒タイムアウト表示: 合格
3. 「今日の記録を見る」で保存した記録を確認: 合格
4. 同じQRの再読取で二重に増えない: 合格
5. 「閉じる」で元の冒険者カードへ戻る: 合格

自動確認でのタイムアウト時間は15.049秒。

実機では、小池先生の試験Aでひまわりが`2/5`から`3/5`へ増え、同じQRを再読取しても1つのままだった。結城先生の試験Bでは、タイムアウト画面の「閉じる」で冒険者カードへ自然に戻った。

最初の簡易QRはスマホカメラで認識されなかったため、標準QR生成方式へ差し替えた。差し替え後は試験A・Bとも読み取れた。
