# 水曜会アプリ 運営手順メモ

2026年6月27日版

## 当日の基本

使用者のスマホは運営が触らない。

使用者はフォームを送るだけ。

運営は自分の端末でフォーム回答を確認し、アプリの管理画面で反映する。

## 参加スタンプ

1. 使用者がトップ画面の「参加する」から参加フォームを送信する。
2. 運営が参加フォームの回答を確認する。
3. アプリ下部の「管理」を開く。
4. 「参加フォーム確認」で名前を入れる。
5. 「今日の参加スタンプを反映」を押す。

注意:

- 参加スタンプは同じ日に1回だけ反映する。
- 名前は操作履歴に残すための確認用。

## 対局記録・先生スタンプ

1. 使用者が先生画面の「対局記録フォームを開く」からフォームを送信する。
2. 運営が対局記録フォームの回答を確認する。
3. アプリ下部の「管理」を開く。
4. 「対局記録フォーム確認」で先生・対局日・ハンデ・勝敗を選ぶ。
5. 「対局記録と先生スタンプを反映」を押す。

注意:

- 反映すると、その先生の対局記録が保存され、先生スタンプが1つ増える。
- 先生の花スタンプが達成済みの場合は反映ボタンが押せない。

## 管理画面の開き方

通常の画面では、下部メニューの「管理」から開く。

見えない時は、URLの最後に `#admin` が付いているか確認する。

例:

`http://127.0.0.1:8787/index.html#admin`

## 困った時

- フォームを送ったのに反映されない: 管理画面で反映ボタンを押したか確認する。
- スタンプ数が違う: 管理画面の「スタンプ数を調整」で直し、「確定して保存」を押す。
- 操作が不安な時: 「直近の操作履歴」を確認する。
- 機種変更や別端末で使う前: 「バックアップを保存」を押しておく。
