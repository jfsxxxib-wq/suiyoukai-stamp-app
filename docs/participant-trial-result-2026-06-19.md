# 参加者試用結果 2026-06-19

## 試用条件

- 参加者：70代男性 1名
- テスト用スタンプ：押していない
- 個人名と対局内容：記録しない

## 観察結果

- 最初から最後まで、操作する場所に迷っていた。
- 文字が小さく、よく見えないという反応があった。
- トップページの「水曜会」、その下の妖精、背景の記号を何度も押していた。
- 先生の写真を何度も見比べ、顔の下側がぼけて見えるという反応があった。
- 「花図鑑」「花集め」という言葉を見て花を探したが、花がどこにあるか分からなかった。
- あちこちを押しながら「どこも押せない」という反応があった。
- フクロウ周辺も何度も押したが、スタンプ操作には進まなかった。

## 分かったこと

- 押せる場所と飾りの区別が視覚的に伝わっていない。
- 文字サイズが、年配の参加者には十分でない。
- 花図鑑への入口と、獲得済み・未獲得の花の位置が見つけにくい。
- 妖精やフクロウは興味を引くが、押した後に何が起きるかが分かりにくい。
- 先生写真は、実際の表示でぼけや切れがないか再確認が必要。

## 改善候補の優先順位

1. 基本文字と下部メニューを大きくし、押せる場所をボタンらしくする。
2. トップページに「まずここを押す」という一つの明確な入口を置く。
3. 花図鑑に「見つけた花」「これから出会う花」の場所が分かる案内を付ける。
4. 妖精、フクロウ、背景など、押せそうに見える装飾の扱いを整理する。
5. 先生写真を実機表示で確認し、ぼけや顔の切れを調整する。
