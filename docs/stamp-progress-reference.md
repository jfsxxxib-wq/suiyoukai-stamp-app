# スタンプ進行見本 保管メモ

## 画面から外した理由

2026-06-22の花図鑑改善で、参加者向け画面から「スタンプ進行見本」を外しました。花図鑑が縦に長くなり、これから置く内容を考えにくくなるためです。

押印、保存、達成判定には使用していない説明用の見本でした。先生ごとの現在回数、花の成長、5回達成モーダルはそのまま動作します。

## 保管する進行内容

| 回数 | 表示メモ |
|---:|---|
| 0回 | つぼみ |
| 1回 | ひとつ咲く |
| 2回 | 色づく |
| 3回 | 花ひらく |
| 4回 | 満開手前 |
| 5回 | 満開達成・妖精出現 |

## 復元するとき

- 削除前のHTMLは、改善分岐のコミット履歴から確認できます。
- `.stamp-progress`、`.stamp-progress-grid`、`.progress-card` などのCSSは、復元しやすいよう現時点では残しています。
- 再表示する場合も花図鑑の常設欄へ戻さず、運営用資料や別の説明画面として検討します。
