# 今日の指導碁カード 本番確認メモ 2026-07-05

目的: GitHub Pages反映後に、対局内容QRで「今日の記録」に本人の対局内容がやさしく表示されるか確認する。

## 現在の状態

- GitHubの元ファイルには、今日の指導碁カード実装が入っている。
- GitHub Pagesの公開URLは、確認時点では古いHTMLを返していた。
- 公開URLが新しい版になるまで、実機確認はいったん待つ。

新しい版の目印:

- `20260705-today-game-record-01`

古い版の目印:

- `20260705-qr-landing-01`

## 確認で使うQR

使う:

- 管理画面で、先生、日付、ハンデ、勝敗を選んで作る「本人スマホに入れるQR」。

使わない:

- 先生ごとの固定QR。
- 受付用 参加QR。

理由:

- 今日の指導碁カードは、本人スマホに入った対局内容を見せるための表示。
- 先生ごとの固定QRは、先生スタンプだけを入れるためのQR。
- ハンデと結果が両方「記録なし」の場合は、対局内容カードとしては出さない。

## 実機確認手順

1. Safariで本番URLを開く。
2. 管理画面を開く。
3. 先生、日付、ハンデ、勝敗を選ぶ。
4. 「本人スマホに入れるQRを作る」を押す。
5. 表示されたQRを、本人スマホ想定の端末で読む。
6. 冒険者カードの「今日の記録」へ着地するか見る。
7. 参加スタンプ、指導碁スタンプの下に、今日の指導碁カードが出るか見る。
8. 同じQRをもう一度読んでも、二重に増えないか確認する。

対局内容QR:

> 今日の指導碁も記録に残りました。

表示確認:

> 冒険者カードの今日の記録を見てください。

## 判定

合格:

- 対局内容QRで、今日の指導碁カードが出る。
- 先生ごとの固定QRでは、先生スタンプだけが出る。
- 同じQRをもう一度読んでも二重に増えない。

保留:

- 本番URLがまだ古い版を返している。

失敗:

- 対局内容QRを読んでも、先生スタンプだけで対局内容カードが出ない。
- ハンデや結果が選んだ内容と違う。

## 実機確認結果

確認日時: 2026-07-05 19:25頃

結果:
