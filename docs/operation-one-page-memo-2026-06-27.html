水曜会アプリ 運営手順メモ

2026年6月27日版

使用者のスマホは運営が触らない。使用者はフォームを送るだけ。運営は自分の端末で確認して反映する。

参加スタンプ

  1. 使用者が「参加する」から参加フォームを送る。
  2. 運営がフォーム回答を確認する。
  3. 下部の「管理」を開く。
  4. 「参加フォーム確認」で名前を入れる。
  5. 「今日の参加スタンプを反映」を押す。

同じ日に反映できる参加スタンプは1回だけ。

対局記録・先生スタンプ

  1. 使用者が先生画面から対局記録フォームを送る。
  2. 運営がフォーム回答を確認する。
  3. 下部の「管理」を開く。
  4. 先生・対局日・ハンデ・勝敗を選ぶ。
  5. 「対局記録と先生スタンプを反映」を押す。

押すと対局記録が保存され、先生スタンプが1つ増える。

管理画面

通常は下部メニューの「管理」から開く。

見えない時はURLの最後に #admin が付いているか確認する。

困った時

  • 反映されない: 管理画面で反映ボタンを押したか確認。
  • 数が違う: 「スタンプ数を調整」で直して保存。
  • 不安な時: 「直近の操作履歴」を確認。
  • 別端末の前: 「バックアップを保存」。
