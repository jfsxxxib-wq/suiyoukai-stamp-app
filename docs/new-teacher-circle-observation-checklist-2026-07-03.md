# 新しい先生の輪 観察期間チェックリスト 2026-07-03

目的: 新しい先生の輪を、すぐ報酬化せず「出会いを記録する場所」として運用し、実際の使われ方を見てから正式な達成ルールや報酬を決める。

## 観察期間の方針

- 既存5人の先生の輪は、水曜会の中心として固定する。
- 新しい先生の輪は、既存5人へ混ぜず別枠で見る。
- 観察期間中は、新しい先生の輪に専用の勲章や称号を追加しない。
- 先生ごとの妖精や称号は既存の花巡りの仕組みとして扱い、「新しい先生の輪」専用報酬とは分けて見る。
- 未使用ストックの花と妖精は、画面に出さない。
- 保存データ形式、先生ID、既存の称号・勲章IDは変えない。

## 報酬化を見直す条件

次のどれかが複数回起きたら、新しい先生の輪を正式な達成ルールや報酬つきの輪にするか会議で見直す。

- [ ] 追加先生の記録が継続して増えている
- [ ] 参加者から「新しい先生の輪も集めたい」という反応がある
- [ ] 追加先生A-Eが一時的ではなく、今後も使う先生枠になった

- 保存データ形式の変更
