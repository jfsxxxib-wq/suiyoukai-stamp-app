# 先生写真の差し替えルール

## 基本方針

- 先生写真は、スタンプ数、押印、履歴、報酬判定から分離した表示素材として扱います。
- 新しい写真へ差し替えても、先生IDと保存データは変更しません。
- 元写真へ、首や肩が消える白いぼかしを焼き込まないようにします。
- 顔全体と肩の一部が自然に見える写真を使用します。

## 推奨する写真

- 正面または少し斜めから撮影した、顔が明るく見える写真
- 縦長または正方形に近い構図
- 目安として横800px以上
- JPGまたはPNG
- 背景が複雑すぎず、人物との境界が自然なもの
- アプリで使用してよいことを確認済みの写真

## 保管方法

1. 現在の写真は削除・上書きせず残します。
2. 新しい写真は `teacher-先生ID-v2.jpg` のように、版が分かる名前で `assets/` へ保存します。
3. `styles.css` の該当する先生1名の画像参照だけを変更します。
4. 他の先生の写真、先生ID、押印データには触れません。

## 確認項目

- 顔の上下が切れていない
- 首や肩が不自然に消えていない
- 拡大しても大きくぼやけない
- スマホで5人の大きさが極端に違わない
- 先生名と写真が一致している
- 写真変更後も押印、履歴、冒険者カードが正常
