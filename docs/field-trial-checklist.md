# 水曜会 実地試用チェック表

## 試用前

- [ ] 管理画面で「バックアップを保存」を押した
- [ ] バックアップファイルがダウンロードにあることを確認した
- [ ] スマホでトップ画面と「次の冒険」が読める

## 参加者1人目

- [ ] 運営確認から参加スタンプを1回押せた
- [ ] 回数と「次の冒険」が正しく変わった
- [ ] 参加者が画面の意味を理解できた

## 参加者2人目（できる場合）

- [ ] 先生を選び、指導碁スタンプを1回押せた
- [ ] 先生詳細の回数へ正しく反映された
- [ ] 操作履歴へ押印が追加された

## 使い終わった後

- [ ] もう一度バックアップを保存した
- [ ] 操作に迷った場所を下へ記録した
- [ ] 文字や画像が見づらかった場所を下へ記録した
- [ ] 運営に負担を感じた操作を下へ記録した

## 気づいたこと

- 操作に迷った場所：

- 見づらかった場所：

- 運営の負担：

- 参加者の反応・感想：

- 次に直したいこと：

## 試用後の方針

試用中は新機能を追加せず、気づいたことだけを記録します。試用後に記録を見て、必要な修正を小さいものから一つずつ行います。
