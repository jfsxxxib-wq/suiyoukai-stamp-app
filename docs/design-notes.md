# デザインメモ

## 全体の方向性

このアプリは、単なるポイントカードではなく「冒険記録帳」です。

大事にしたい雰囲気:

- 水曜会に参加する楽しみ
- 御朱印のような記念感
- 花図鑑を集めるワクワク感
- 冒険者カードが育つ感じ
- 自分の思い出を綴るメモ感

## 画面の考え方

- 最初の画面は、説明ページではなく、実際に使う画面にする
- スマホで見やすいことを優先する
- ボタンやカードの文字は詰め込みすぎない
- 先生写真を花や妖精で隠しすぎない
- 達成演出は先生カード内に重ねず、別モーダルで出す

## 冒険者カード

表示したいもの:

- 冒険者名
- 称号
- 勲章
- 花図鑑達成率
- 達成した妖精
- スタンプ数は小さめ

イメージ:

- ポイントカードではなく、冒険記録帳
- 自分の旅の成果がまとまる場所
- 勲章と称号は、あとから本番デザインを整える

## 先生画面

先生画面で大事なこと:

- 先生名がわかりやすい
- 先生写真が主役として見える
- どの花・妖精に対応しているかがわかる
- 5回達成は別画面で祝う
- 先生の輪一巡と妖精達成を混同しない

## 素材メモ

主な素材:

- 花スタンプ: コスモス、菖蒲、椿、ひまわり、紫陽花、桜、ダリア、梅
- 妖精: アポロン、菖蒲、椿、ひまわり、紫陽花、桜、ダリア
- 勲章: 参加スタンプ、先生の輪
- 先生写真: 5人分
- 地図: 湘南風の水彩マップ

## 注意したいこと

- 画像ファイル名とルール表の対応をそろえる
- 新しい花を追加するときは、花素材と妖精素材をセットで確認する
- 画面内に古いアポロン固定画像が残っていないか確認する
- 先生ごとの5回達成は、対応する花の妖精を出す
- ブラウザ表示が古い時は `Ctrl + F5` で確認する
