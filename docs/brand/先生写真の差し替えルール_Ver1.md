# 水曜会アプリ
## 先生写真の差し替えルール Ver.1

### 目的

先生ごとの個性を残しながら、一覧で見たときに統一感があり、やさしく見やすい表示にする。
写真表示でも、水曜会らしさの基準である次の3点を保つ。

- やさしい
- 品がある
- 初めての方にも開かれている

---

## 基本方針

- 統一感 7 / 個性 3
- 写真は先生ごとの雰囲気を残してよい
- ただし、一覧表示でバラつきすぎないように、次の要素をなるべくそろえる
  - 顔の大きさ
  - 目線の高さ
  - 頭上の余白
  - 肩や胸元の見える量
  - 明るさ
  - 背景の強さ

---

## 写真ルール

### 1. 画像の向きと比率

- 縦長の人物写真を基本とする
- 写真表示枠の縦横比は固定する
- 画像を無理に引き伸ばさない
- 顔が切れないことを優先する

---

### 2. 顔の大きさ

- 顔が小さすぎないようにする
- 顔が写真全体の中心的な印象になる大きさにそろえる
- 先生ごとの差は多少あってよいが、極端な差は避ける

目安：

- 頭の上に少し余白を残す
- 肩から胸元付近まで見える程度にする
- 顔だけが大きくなりすぎないようにする

---

### 3. 顔の位置

- 顔はやや上寄り中央を基本とする
- 左右どちらかに大きく寄りすぎないようにする
- カードごとに目線の高さが大きくズレないようにする

---

### 4. 余白

- 頭上には少しだけ余白を確保する
- 顔の左右に極端な空きが出ないようにする
- 下側は肩から胸元が少し見える程度でよい
- カードごとの余白感をできるだけそろえる

---

### 5. 背景

- 背景は、うるさすぎないものを優先する
- 単色、やわらかい壁面、落ち着いた室内背景などが望ましい
- 背景色が強い場合は、必要に応じて次の調整を行う
  - 少し明るくする
  - 彩度を少し抑える
  - コントラストを弱める
  - 写真全体をやわらかく見せる
- 背景が強すぎて、他の先生写真より目立ちすぎないようにする

---

### 6. 明るさと色味

- 写真全体は明るめで自然な色を基本とする
- 暗すぎる写真は軽く補正してよい
- 黄み、赤み、青みが強すぎる場合は軽く補正してよい
- 肌色が不自然にならないようにする
- 写真ごとの明るさの差をできるだけ小さくする

---

### 7. 表情と雰囲気

できれば次の印象がある写真を優先する。

- 穏やか
- 親しみやすい
- 清潔感がある
- 初めて見る人が安心しやすい

真顔でもよいが、極端に怖く見える写真は避ける。

---

### 8. 服装

- スーツ、ジャケット、きれいめな服装は相性がよい
- カジュアルな服装でも、全体の雰囲気に合っていれば使用してよい
- 柄や色が強すぎる場合は、他の写真との調和を確認する

---

## 表示ルール

### 9. 写真枠

- 写真枠の大きさは全カードで統一する
- 角丸の大きさを統一する
- 枠線の太さと色を統一する
- 枠線はやわらかい色を使用する
- 写真は先生の印象が伝わる十分な大きさにする

---

### 10. 名前とのバランス

- 先生名の文字サイズと位置を全カードで統一する
- 名前の視認性を写真より優先する
- 写真の存在感が強すぎて、名前が埋もれないようにする
- 段位を含めた名前の行間と文字間隔をそろえる

---

### 11. 花情報との関係

- 「いま育てている花」の情報が読みやすいことを優先する
- 写真が大きくても、文字情報の余白を圧迫しないようにする
- 名前、花名、先生詳細ボタンの間隔をそろえる

---

## 実装方針

### 推奨CSS

```css
.teacher-photo {
  width: 100%;
  aspect-ratio: 4 / 5;
  object-fit: cover;
  object-position: 50% 35%;
  border-radius: 20px;
}
```

### 実装上の注意

- 写真表示枠は固定サイズにする
- `object-fit: cover` を使用する
- 写真ごとに顔の位置が異なる場合は、個別に `object-position` を調整できるようにする
- 写真データごとに位置調整値を持たせてもよい
- 一覧で見たときに、顔の大きさと目線の高さが大きくズレないようにする

---

## 写真ごとの個別調整例

```js
const teachers = [
  {
    name: "徐 文燕 三段",
    photo: "/images/teachers/jo-bunen.jpg",
    photoPosition: "50% 30%"
  },
  {
    name: "大西 竜平 七段",
    photo: "/images/teachers/onishi-ryohei.jpg",
    photoPosition: "50% 38%"
  }
];
```

```jsx
<img
  className="teacher-photo"
  src={teacher.photo}
  alt={`${teacher.name}の写真`}
  style={{ objectPosition: teacher.photoPosition }}
/>
```

## 差し替え時の確認チェックリスト

写真を入れ替えるたびに、次を確認する。

- 顔が小さすぎない
- 顔や頭が不自然に切れていない
- 目線の高さが他のカードと大きくズレていない
- 頭上の余白が広すぎない
- 肩や胸元の見える量が極端に違わない
- 背景が強すぎない
- 写真が暗すぎない
- 肌色が不自然ではない
- 写真だけが目立ちすぎていない
- 先生名が読みやすい
- 花名が読みやすい
- 先生詳細ボタンが埋もれていない
- 全体としてやさしく、品のある印象になっている

---

## デザイン判断の優先順位

迷った場合は、次の順で判断する。

1. 先生の顔が見やすい
2. 一覧で統一感がある
3. 水曜会らしいやさしさと品がある
4. 先生ごとの個性が伝わる

---

## 最終方針

- 全員を完全に同じ構図にそろえる必要はない
- 写真ごとの個性は残してよい
- ただし、別々のアプリから集めた写真のように見えるほどの差は避ける
- 一覧では自然にそろって見え、個別に見ると先生らしさが伝わる状態を目指す

---

## 今回の見本に対する調整方針

### 徐文燕三段

- 現在の正面向きで明るい印象はよい
- 顔の大きさと頭上の余白を基準候補にする
- 背景が淡いため、水曜会全体の雰囲気と合わせやすい

### 大西竜平七段

- 先生らしい自然な表情と動きがあり、個性が伝わる
- 黄色い背景が強いため、必要に応じて彩度または明るさを軽く調整する
- 顔の位置と大きさを左側の写真に少し近づける
- 個性は残しつつ、背景だけが強く見えすぎない状態を目指す
