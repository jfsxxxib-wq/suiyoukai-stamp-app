# 冒険日誌・組み合わせ神社 実地確認チェックリスト

2026年7月1日版

目的は、新機能を増やすことではなく、今ある書庫・冒険日誌・組み合わせ神社が「水曜会の1回分」で自然に使えるかを見ることです。

確認中は、気づいたことだけ記録します。試用中に直さず、あとで小さく直します。

## 確認前

- [ ] ブラウザを `Ctrl + F5` で再読み込みした
- [ ] トップ画面の「次の冒険」が読める
- [ ] 下部の「書庫」を押せる
- [ ] 下部の「花図鑑」「冒険者カード」「書庫」の文字が読める
- [ ] 画面が横にはみ出していない

## 書庫と冒険日誌

- [ ] 書庫を開いた時、冒険日誌が見える
- [ ] 冒険日誌の本とハリネズミが狭く重なっていない
- [ ] ハリネズミの吹き出しが出る
- [ ] 吹き出しの一言が長すぎない
- [ ] 吹き出しが「ひとこと残す」ボタンの邪魔をしていない
- [ ] 冒険日誌の記録が4件前後で読みやすい
- [ ] 神社由来の記録、手入力のひとこと、達成記録が見分けられる
- [ ] 日誌を閉じた時、記録欄と吹き出しも一緒に閉じる
- [ ] 日誌をもう一度開いた時、記録欄と吹き出しが戻る

## ひとことメモ

- [ ] 「ひとこと残す」を押すと入力欄が出る
- [ ] 短いひとことを入力できる
- [ ] 「残す」を押すと日誌に追加される
- [ ] 追加後、ハリネズミの一言が変わる
- [ ] 空欄で「残す」を押しても画面が壊れない
- [ ] 「今日は書かない」を押しても、記録を書く義務感が出ない

## 組み合わせ神社から日誌へ

- [ ] トップ画面から組み合わせ神社へ入れる
- [ ] 「組を作る」「対戦を決める」「結果を見る」の流れが読める
- [ ] 見本を入れて組み合わせを作れる
- [ ] 「今の組み合わせを残す」を押せる
- [ ] 保存後、書庫の冒険日誌に短い出来事が増える
- [ ] 神社から来た記録が `⛩` の行として見分けられる
- [ ] 保存後に再読み込みしても、保存した組み合わせと日誌メモが残る

## スマホで見るところ

- [ ] 文字が小さすぎない
- [ ] ボタン同士が近すぎない
- [ ] 日誌欄のスクロール量が多すぎない
- [ ] ハリネズミが小さすぎず、何の絵か分かる
- [ ] 妖精用の右下スペースがボタンや文字にかぶっていない
- [ ] 書庫全体が縦に長すぎて迷わない

## 壊れやすい操作

- [ ] 日誌を何度も開閉しても表示が崩れない
- [ ] 「ひとこと残す」を何度か押しても入力欄が変に増えない
- [ ] 神社で保存を連続して押しても画面が壊れない
- [ ] 書庫、神社、トップを行き来しても戻れる
- [ ] 何も記録がない状態でも、空の案内が自然に見える
