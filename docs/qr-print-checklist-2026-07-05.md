# QR印刷確認チェックリスト 2026-07-05

状態: 紙運営用

目的: 老眼の人、手早く操作する人、ゆっくり確認する人でも、当日QRを迷わず読める紙面にする。

## 1. 受付用参加QR

- 紙面の見出しが「受付用 参加QR」になっている。
- QRのまわりに余白がある。
- 参加者がスマホを近づけても読み取りやすい大きさになっている。
- 日付が当日であることを確認した。
- 1回読んだ後、冒険者カードの「今日の記録」へ着地する。
- 同じQRをもう一度読んでも、参加スタンプは増えない。

## 2. 先生ごとの固定QR

- 先生名が大きく読める。
- 日付が読める。
- QRと先生名の対応が間違っていない。
- 先生ごとに1枚ずつ分かれている。
- 先生席に置いた時、紙の上下が分かる。
- 読み取り後、冒険者カードの「今日の記録」へ着地する。
- 1人目、2人目、3人目の花が順番に見える。

## 3. 対局内容QR

- 先生、日付、ハンデ、勝敗を選んでから作る。
- 本人スマホに入れる時だけ使う。
- 読み取り後、冒険者カードの「今日の記録」へ着地する。
- 運営PCだけで確認したい時は、本人スマホ用QRと混ぜない。

## 4. 当日の禁止メモ

- 前日確認用QRを置きっぱなしにしない。
- 先生名が分からないQRだけを置かない。
- 受付用参加QRと先生QRを同じ山にしない。
- 本人スマホに入ったQR記録を、運営PCから戻せると思わない。

## 5. 合格ライン

- 受付の人が、説明なしで参加QRを見つけられる。
- 先生名を見て、正しい先生QRを選べる。
- iPhoneカメラで一発または数秒以内に読める。
- 読み取り後に「今日の花が入りました。」と言える画面へ着地する。
