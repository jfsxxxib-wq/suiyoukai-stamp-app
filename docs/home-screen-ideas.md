# ホーム画面アイデア

## 江ノ電風の大きな電車背景 2026-06-28

組み合わせ神社の導入演出を調整している途中で、背景の湘南マップが偶然よい構図で表示された。

- 右側に江ノ電風の緑の電車が大きく出る。
- 左側にカードやボタンを置ける余白がある。
- 水彩の海、花、線路があり、冒険記録帳の入口画面に合いそう。
- ホーム画面を作る時は、背景画像を全体表示ではなく `cover` で使い、右側の電車が大きく見える位置に寄せる。
- UIは左側に置き、電車と花を背景の主役として残す。
